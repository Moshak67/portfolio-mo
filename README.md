## Run My Portfolio Locally

Clone the project

```bash
  git clone https://github.com/Moshak67/portfolio-mo.git
```

Go to the project directory

```bash
  cd portfolio-mo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
